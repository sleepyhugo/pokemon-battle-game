# Mini Pokémon Battle Game (Java Console Project)

This is a small, fun Java console game where you pick a starter Pokémon and battle against a CPU opponent.  
It’s simple, beginner-friendly, and built to practice Java fundamentals like classes, enums, arrays, methods, randomization, and basic game logic.

---

## Features

- Choose from **three starter Pokémon** (Fire, Water, Grass)
- Each Pokémon has:
    - A type (FIRE / WATER / GRASS / NORMAL)
    - Unique moves with different power levels
    - HP that decreases as the battle continues
- **CPU randomly picks a Pokémon and attacks back**
- **Turn-based battle system**
- **Type effectiveness** (super effective, not very effective)
- Randomized damage for more dynamic gameplay
- Clear, beginner-friendly code structure

## How to Run

1. Make sure you have **Java 17+** installed.
2. Clone the repo:

```bash
git clone https://github.com/sleepyhugo/pokemon-battle-game.git
```

3. Navigate into the folder:
cd pokemon-battle-game/src

4. Compile and run:  
javac *.java  
java BattleGame  

That’s it - the game launches in the console!
